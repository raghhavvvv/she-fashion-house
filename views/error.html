Application Error

There was an issue starting the She Fashion House application. This usually happens when the server process cannot start properly.

Troubleshooting Steps:

  • Make sure no other application is using port 3000
  • Check that all files are properly extracted
  • Try running the application as administrator
  • Ensure your antivirus isn't blocking the application

If the problem persists, please contact support with the error details from the console.
